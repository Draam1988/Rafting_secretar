# RaftSecretary

Офлайн-инструмент секретаря соревнований по рафтингу.

Что уже есть:
- настройки соревнования;
- судьи;
- команды и составы;
- спринт;
- H2H;
- слалом;
- длинная гонка;
- протоколы результатов.

## Локальный запуск

```bash
python3 server.py
```

Потом открыть:

- [http://127.0.0.1:8000](http://127.0.0.1:8000)

## Windows-сборка

Для Windows в проекте есть:
- `launcher.py`
- `build_windows.bat`
- GitHub Actions workflow:
  - `.github/workflows/windows-build.yml`

После сборки готовая папка будет:

- `dist\\RaftSecretary`

Тестерам нужно передавать всю эту папку целиком, а не один `.exe` отдельно.
