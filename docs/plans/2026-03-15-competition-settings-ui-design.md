# Competition Settings UI Design

Дата: 2026-03-15

## Контекст

Текущий экран `Соревнование` перегружен постоянным показом всех дисциплин и всех категорий. Для секретаря это лишний шум, потому что:
- часто используется ограниченный набор дисциплин;
- часто используется 1 возрастная группа, реже 2;
- настройки соревнования вводятся один раз и потом служат опорой для рабочего стола.

## Цель

Сделать экран `Соревнование` компактным и рабочим:
- основные данные видны сразу;
- блоки `Дисциплины` и `Категории` скрыты по умолчанию;
- в заголовках этих блоков видна короткая сводка;
- на дашборде показываются только те дисциплины, которые реально включены.

## Решение

### Экран `Соревнование`

Оставить 3 смысловых блока:
- `Основные данные`
- `Дисциплины`
- `Категории`

Блоки `Дисциплины` и `Категории` сделать раскрывающимися секциями.

В свернутом виде показывать короткую сводку:
- `Дисциплины: N выбрано`
- `Категории: N выбрано`

В раскрытом виде:
- `Дисциплины` показывают чекбоксы по доступным дисциплинам;
- `Категории` показывают чекбоксы по доступным комбинациям.

### Дашборд

Дашборд должен строиться динамически:
- базовые блоки всегда видны:
  - `Соревнование`
  - `Судьи`
  - `Команды`
  - `Многоборье`
  - `Экспорт`
- дисциплинарные блоки показываются только если дисциплина включена в настройках соревнования:
  - `Спринт`
  - `Параллельный спринт`
  - `Слалом`
  - `Длинная гонка`

## Что не входит в эту правку

- отдельные даты по каждой дисциплине;
- мастер создания соревнования;
- блок `Судьи`;
- экспорт документов.

## Критерий успеха

После правки:
- экран `Соревнование` выглядит компактнее;
- `Категории` и `Дисциплины` не забивают экран;
- дашборд не показывает лишние дисциплины.
